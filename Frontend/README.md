# AI Interview Prep Frontend

React + Vite client for the AI Interview Prep Platform.

## Scripts

```bash
npm install
npm run dev
```

Other scripts:

- `npm run build` builds the production bundle.
- `npm run preview` serves the production build locally.
- `npm run lint` runs ESLint.

## Environment

Set `VITE_API_URL` when the backend is not running at `http://localhost:3000`.

```env
VITE_API_URL=http://localhost:3000
```

See the root `README.md` for the full project setup and API reference.
